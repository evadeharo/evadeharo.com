# React + TypeScript + Vite

This repo is a simple installation based on React + TypeScript + Vite + Tailwind init to install Three fiber and do tests.

To maintain order in styling we are using: 
- [Prettier plugin for Tailwind](https://prettier.io/docs/configuration.html)
- [Cva](https://cva.style/docs)

This repository uses yarn. 
Just run “yarn dev” to open a local port.
